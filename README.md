As an alternative to supervised approaches, this project contains three prompt-based classifiers for the detection of contradictions and inconsistencies between regulatory texts. The accompanying paper can be found [here](https://hicss.hawaii.edu/). The description of the collection, preprocessing, and annotation of the data used is detailed in the paper [Detection of Contradictions and Inconsistencies in German Regulatory Documents](https://ieeexplore.ieee.org/abstract/document/10692679?casa_token=XZbIx1qqiCUAAAAA:T2CxqFoLAvT6902_9aU3EEdSAFfRsHQPDgryu_HORDXIRk8x7w4X8lozdjaxJc_wDlojc5Wlv7cd4w).
